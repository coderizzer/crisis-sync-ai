
# 🚨 CrisisSync AI  
### Real-Time AI-Powered Emergency Intelligence System

---

## 📌 Overview

**CrisisSync AI** is a real-time emergency response assistant that leverages AI to transform unstructured incident reports into structured, actionable intelligence.

It enables faster, smarter, and more coordinated responses during critical situations by automatically classifying emergencies, assigning severity levels, and maintaining a live incident dashboard.

---

## ❗ Problem Statement

Emergency communication is often:
- Unstructured and ambiguous  
- Delayed in processing  
- Dependent on manual triage  

This leads to:
- Slower response times  
- Misclassification of incidents  
- Inefficient resource allocation  

---

## 💡 Our Solution

CrisisSync AI introduces an **AI-driven classification layer** that:

- Interprets natural language emergency inputs  
- Identifies incident type (fire, medical, threat, etc.)  
- Assigns severity (low / medium / critical)  
- Extracts location context  
- Displays incidents in a real-time dashboard  

---

## ⚙️ Core Features

- 🧠 **AI-Powered Classification** (Google Gemini)  
- 🚨 **Critical Alert Detection Mode**  
- 🎤 **Voice Input (Speech-to-Text)**  
- 📊 **Live Incident Dashboard**  
- ⚡ **Instant JSON Structuring of Reports**  
- 🎯 **Minimal, Fast, and Scalable UI**

---

## 🔄 Workflow

1. User reports an emergency via text or voice  
2. Request is sent to backend API  
3. Gemini processes and classifies the input  
4. Returns structured JSON:
   - Type  
   - Severity  
   - Location  
   - Summary  
5. UI updates instantly with alerts and incident logs
